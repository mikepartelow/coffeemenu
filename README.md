# CoffeeMenu

I wonder what's for sale at my favorite coffee roasters?

## Build

    go build ./cmd/menu

## Run

    # see the menu
    ./menu

    # see the menu with boring MarkDown
    ./menu -boring

    # see who is offering Rwanda coffee right now
    ./menu -csv | grep -i rwanda
